Если не проходит активация Windows, например активация ругается на попытку активировать Windows не в своем географическом регионе, необходимо в командной строке набрать команду:


slui.exe 4


И затем, после ввода логина-пароля Администратора, нужно выбрать регион "Россия". Потом нажать "Далее", бодет отображен номер телефона, по которому можно позвонить в Microsoft для автоматической регистрации через телефон. Далее следовать инструкцией мастера на экране.
