В методике InterView это делается через модель. То есть, для вида выясняется модель, и у этой модели выясняется QModelIndex для указанных строки и столбца.


Пример получения QModelIndex первого левого элемента для любого вида:


QModelIndex firstIndex = listView->model()->index( 0, 0 );


Этот код может работать для QListView, QTreeView, и для прочих видов, основанных на таблицах.
