У меня несколько проектов, и несколько серверов на разных континентах. Однажды на одном из проектов возникла проблема с нагрузкой (с самой виртуалкой проблем небыло), я узнал о проблемах с IO диска из алертов что linode рассылает при их возникновении.

Там есть много функций, таких как:

— NodeBalancers — крутая штука из описания, но я пока не восспользовался

— Additional RAM — можно добавить оперативы

— Additional Storage — дополнительный диск

— Additional Transfer — дополнительный лимит по трафику, т.к. linode устанавливает всем ограничения

— Additional IP Addresses — ну а это дополнительный ip

В панели управления есть ещэ много всяких штук, типа мониторинга, бэкапов, клонирования виртуалок и API

Поддерживаю насчет linode, у самого проекты там висят, никаких нареканий.
