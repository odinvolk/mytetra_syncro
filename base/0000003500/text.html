Тут написано только как подключить диск телефона через IRDA Usb.

У меня есть данное устройство - IRmate 410W. Без особых проблем настроился и работает. Делал следующее:

1. установил пакет irda-utils

2. настроил файл /etc/default/irda-utils (ENABLE="true",AUTOMATIC="true",DISCOVERY="true",DEVICE="irda0",DONGLE="none",SETSERIAL="/dev/ttyS1")

3. проверил модули modprobe ircomm, modprobe ircomm-tty

4. перезапустил irda-utils

5. проверил: sudo irdadump (подключал телефон siemens cx65)

6. установил obexfs и прописал исполняемый файл на монтирование с содержанием: obexfs -i /media/mobile

7. прописал файл на демонтирование с содержанием: umount /media/mobile
