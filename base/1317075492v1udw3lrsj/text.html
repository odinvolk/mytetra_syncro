Timidity в Linux


Единственная программа, которая заработала - timidity. Через него MIDI файлы сразу заиграли.


Пользоваться timidity можно так. Вначале запускается MIDI-демон. Он запускается путем вызова команды timidity без параметров. После запуска можно прерить наличие демона в процессах. Если он есть, то половина дела сделано:



$ timidity

Прослушать MIDI можно через ту же программу timidity, указав ей MIDI-файл:


$ timidity music.mid

Format: 1 Tracks: 11 Divisions: 384

Track name: Beznih
