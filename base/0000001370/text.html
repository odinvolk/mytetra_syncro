На сайте http://tv-here.ru канал просматривается через вызов JavaScript-функции

mogulus('channel=docsru')

Выглядит она так:

function mogulus(url)

{

document.write('

<object width="600" height="475" id="1" classid="clsid:D27CDB6E-AE6D-11cf-96B8-444553540000">
