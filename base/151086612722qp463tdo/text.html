Потыкал базовый функционал. Вполне прилично, все понятно.

Искаропки поддерживает несколько организаций, внутри них список дел. Карточка дела может иметь обсуждение. К карточке можно прикреплять рисунки и любые бинарные файлы. Карточке можно назначать текстовые метки (теги). Карточки распределяются по категориям типа «Сделать», «В работе», «Сделано». Категории представляются в виде вертикальных столбцов. Карточки перетаскиваются между категориями драганддропом, но есть возможность сделать перенос через кнопочки - указываешь действие move, целевую категорию, и карточка перепрыгнет куда нужно. Можно сортировать карточки по высоте, то есть по важности. Карточки можно помечать цветом.



К сообщениям (в обсуждении карточки) прикреплять картинки нельзя, вот это непонятно почему. Есть всякие эмоджи, а прикрепляемых картинок нет.



Внизу страницы панель быстрого доступа. Есть кнопка просмотра текущей активности пользователей и последних действий в системе.

Претензия только к размещению кнопок интерфейса. Все неочевидно, пока найдешь и поймешь как какое-то действие сделать, мозго сломаешь. Ну это общая проблема людей, выросших на вебдваноль и GTK-интерфейсах, они те еще проектанты.
