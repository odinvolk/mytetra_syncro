Текстовое содержание тега можно изменить при помощи метода text(). В параметре метода указывается новое текстовое содержимое:


$('#text').text('У элемента с id=text установится это новое содержимое.');


Полный пример:


<html>

<head>

<script src="http://code.jquery.com/jquery-latest.js"></script>

<script>

$(document).ready(function(){

$("#btn").click(function ()

{

$('#text').text("Стало");

});

});


</script>

</head>

<body>

<div id="text">Было</div><br />

<button id="btn">Кликни!</button>

</body>

</html>
