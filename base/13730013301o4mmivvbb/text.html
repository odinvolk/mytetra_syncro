На 05.07.2013 компания МТС рекомендует следующие настройки:

Настройки Интернет GPRS/EDGE:

  • Название профиля/Profile name: MTS Internet
  • Канал данных/Data bearer: GPRS
  • Точка доступа/APN: internet.mts.ru
  • Имя пользователя/User name: mts
  • Пароль/Password: mts

Однако уже месяц как интернет с этими настройками не работает.

На сайте МТС можно найти также настройки телефона в качестве модема. Впринципе, они те же самые:

  • Название профиля/Profile name: mts-gprs web
  • Канал данных/Data bearer: GPRS
  • Точка доступа/APN: internet.mts.ru
  • Первичный DNS: 000.000.000.000
  • Вторичный DNS: 000.000.000.000
  • Имя пользователя/User name: mts
  • Пароль/Password: mts
