GoGetLinks.net - биржа GoGetLinks для приоритетного продвижения в Яндексе. Успешный проект, с жесткими правилами но хорошими выплатами.


  • GetGoodLinks.ru - русскоязычная версия биржи GetGoodLinks для продвижения в Гугле
  • GetGoodLinks.net - англоячычная версия биржи GetGoodLinks для продвижения в Гугле


- эти сервисы появились позднее, считаются менее качественными для оптимизаторов. Но выплаты, говорят, тоже хорошие.


Разница между ГоГетЛинкс и ГоуГудЛинкс только в точной фильтрации. Для первой важен тИЦ, а для второй важен PR, ну и в каждой важен процент проиндексированных страниц.


Биржа getgoodlinks предоставляет техсредства вебмастеру по детектингу на сайте страниц с pr и затем продажа заданий на них через обработку заявок.


Оптимизатору есть опция создать кампанию - мол готов платить Х за страницы с pr Y по тематике Z и затем просев заявок.


В gogetlinks вы через биржу никак страницы с pr не купите. Нужны ли вам страницы с pr - отдельный вопрос, я пейшу о технических средствах. Т.е. "где лучше работать" - тут я увы посоветовать ниче не могу ::разводит руками::
