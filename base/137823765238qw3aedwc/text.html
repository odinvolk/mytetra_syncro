Qualcomm Product Support Tools со стандартным micro-USB кабелем прекрасно работают:

http://www.x-drivers.ru/catalog/flash/mobile_phones/companies/qualcomm/models/qpst/12051.html

Нужно перевести телефон в QUALCOMM USB Mode нажав *#782872# и ОК.

Потом установить драйвер:

http://www.softhour.com/download/mobile/qualcomm-usb-driver

Можно делать бэкап, шить и еще бог знает что..

Все проделал с micro-USB кабелем от Nokia.

Через PST EFS explorer доступна файловая система телефона.

В каталоге \Registry, например, куча файлов конфигураций.

Большое поле для деятельности.. Главное не окирпичить телефончик.
