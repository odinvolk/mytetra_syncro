fuse: bad mount point - Transport endpoint is not connected


Столкнулся с проблемой: пытаюсь смонтировать sshfs, а в результате получаю вот такую ошибку


fuse: bad mount point - Transport endpoint is not connected


Решается такая проблема след. образом:


  • ps -aux | grep ssh
  • kill процесс sshfs
  • размонтировать и смонтировать нужный каталог снова: umount -l /mnt/foobar
  • смонтировать sshfs снова


Либо можно добавить след. опции при монтировании sshfs:


-o reconnect -C -o workaround=all
