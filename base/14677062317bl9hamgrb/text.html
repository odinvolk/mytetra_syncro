Если Linux-машина получает IP по DHCP, то скорее всего, файл /etc/resolv.conf будет постоянно переписываться при обновлении данных с DHCP, и поэтому напрямую править этот файл нельзя.


В современных дистрибутивах файл resolv.conf создается автоматически и править вручную его не нужно. Для получения результата нужно добавить необходимые изменения в файл:


/etc/resolvconf/resolv.conf.d/head


Данные, которые размещены в нем, будут автоматически вставлены в файл /etc/resolv.conf.
