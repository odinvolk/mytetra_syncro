Делай так.

1. Добудь из любого пакета файл alfresco.war

2. Установи на свой линух sun java (x32), tomcat (x32), MySql (x32) или PostgreSQL (x32), ooffise (x32) и т.п.

3. Положи war файл в папку $$$/tomcat/webapps/

4. Из любого unux пакета alfresco возьми папку $$$/tomcat/shared/ и положи ее в свою систему в $$$/tomcat/ исправь пути и другие настройки alfresco.

5. Запусти tomcat

6. Ищи файл alfresco.log в нем должна быть информация о загрузке системы. если файл не появляется то нужно задать путь к нему в файле

$$$/tomcat/webapps/alfresco/WEB-INF/classes/log4j.properties (папка $$$/tomcat/webapps/alfresco/ создается при запуске tomcat (alfresco) )

7. Читай ошибки в alfresco.log и смотри что еще не хватает.

Удачи!

P.S.

Если будешь использовать MySql то возможно понадобится переложить файл mysql-connector-*-bin.jar из пакета alfresco в свою систему.

Возможно tomcat`у нужно будет сказать TOMCAT6_SECURITY=no или что-то подобное... а также вероятно его нужно будет пересадить на порт 8080
