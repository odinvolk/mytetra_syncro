Вопрос:


Полгода назад я работал с репозиторием, делал какие-то изменения, но меня отвлекли и я не успел их закоммитить. Сейчас вернулся к проекту, и смотрю текущий статус:

git status


Вижу около пятнадцати файлов в статусе modified. Перед тем, как сделать коммит, мне нужно детально посмотреть, что было изменено. Какой командой можно посмотреть изменения, внесенные в рабочей директории, по сравнению с последним коммитом?



Ответ:


Посмотреть сразу все изменения:


git diff HEAD


Посмотреть изменения только одного файла:


git diff имяФайла
