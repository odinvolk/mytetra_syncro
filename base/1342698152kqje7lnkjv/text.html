Описание программы лицензирования FPP

Описание варианта лицензирования

Коробочные версии (FPP - Full Packaged Product, полностью упакованный продукт) предназначены для организаций или частных пользователей и доступны в розничной продаже, в интернет-магазинах или у партнёров.

Для коробочных версий права на использование продукта регулируются условиями Лицензионного соглашения конечного пользователя (EULA - End User License Agreement). В EULA оговариваются такие важные аспекты использования, как количество установок по одной лицензии, возможность переноса продукта на другой ПК, необходимость активации и наличие технической поддержки Microsoft, а также перечисляются необходимые компоненты для подтверждения легальности ПО.

Состав продукта

Обычно покупатель в составе коробочной версии получает все необходимые компоненты для установки и использования продукта, а именно: лицензионное соглашение, сертификат подлинности, дистрибутив с программным продуктом, регистрационную карточку и документацию в печатном виде.

Состав коробочной версии может сильно отличаться от продукта к продукту, от версии к версии. Обязательными составляющими являются только коробка, сертификат подлинности на ней и Лицензионное соглашение конечного пользователя (EULA), которое часто представлено в электронном виде.

Для примера приведем компоненты двух наиболее распространенных программных продуктов.

Компоненты Windows XP:

упаковка продукта с сертификатом СОА, наклеенным на верхний торец коробки;

информационный носитель (диск с экземпляром продукта и голограммой);

печатная версия лицензионного соглашения.

Компоненты Windows Vista:

упаковка продукта с наклейкой сертификата COA с штрих-кодом на оборотной стороне футляра для DVD;

информационный носитель (диск с экземпляром продукта и голограммой) в футляре.

Сертификат подлинности (СОА) наклеен на коробку и является подтверждением лицензионных прав пользователя, если иное не предусмотрено в EULA. На СОА указано наименование продукта. См. также примеры СОА в приложении.

Для большинства продуктов 25-тизначный ключ для установки продукта (Product Key) размещен на наклейке внутри коробки.

Проверка подлинности продукта

Для облегчения проверки подлинности продукта, а также для возможной дальнейшей передачи продукта другому лицу, пользователям коробочных версий рекомендуется сохранять все компоненты приобретённого продукта: носители и печатные материалы, лицензионное соглашение и сертификат подлинности, а также подтверждающие платёжные документы.

Проверить подлинность копии ОС Windows XP можно через онлайновый процесс проверки подлинности на сайте Microsoft, выбрав кнопку «Проверка подлинности Windows» справа вверху страницы: http://www.microsoft.com/genuine/default.aspx?displaylang=ru

Проверка подлинности ОС Windows Vista осуществляется в процессе активации при первоначальной установке системы, а также может быть предложена пользователю в процессе использования.

Расширенная информация о проверке подлинности: http://www.microsoft.com/resources/howtotell/default.aspx?displaylang=ru

Подтверждение легальности коробочного продукта

Подтверждением лицензионных прав пользователя является Сертификат подлинности (СОА), если иное не оговорено в Лицензионном соглашении конечного пользователя.

Сертификат подлинности (СОА) наклеен на коробку, на нем указано наименование продукта.
