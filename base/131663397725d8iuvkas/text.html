Настроить работу клавиши Ins в TotalTerminal можно с помощью программы Keyboard Maestro + настройки в самом терминале.

Решение следующее.

1. В Keyboard Maestro сделать отдельную группу, например "Terminal Macro Group". Установить в ее свойствах

Avalable in the following applications: Терминал.app

2. Настроить Keyboard Maestro так, чтобы при нажатии клавиши Help (это и есть клавиша Ins) генерировалось нажатие клавиши F12. (Настроить на F13-F20 почему-то не получилось). Настройку делать только в группе "Terminal Macro Group". То есть, нужно создать макрос, например, "Terminal Ins" и установить:

Triggered by any of the following

This hot key

Help

Will execute the following actions

Type the F12 key stroke

Simulate key stroke: F12

3. В программе TotalTerminal нужно зайти в "Настройки", выбрать вашу тему -> вкладка "Настройки" -> "Клавиатура", далее выбирать клавишу F12 и присвоить ей Esc-последовательность \033[2~.
