Здесь приводятся оптимальные параметры конвентирования из файла формата MTS, в котором находятся видеоданные в формате HD PS 50p.


Конвентирование имеет смысл проводить в H264. При конвентировании можно понизить разрешение с 1920x1080 на 1440x810 для возможности комфортной работы в программах видеомонтажа на Intel Core 2 Duo. Частоту кадров можно оставить прежнюю в 50 кадров/сек.


#!/bin/sh


echo "Begin: " `date` # Время начала кодирования на экран

date +'Begin: %c' > date.txt # Время начала кодирования в файл


infile="00006.MTS"

outfile="rotate_50.avi"


mencoder ${infile} -o ${outfile} -oac copy -noskip -ovc x264 -x264encopts bitrate=12000:subq=7:frameref=10 -vf scale=1440:810
