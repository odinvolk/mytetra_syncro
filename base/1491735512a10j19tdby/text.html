Для unix вводим следующие строки в консоли:


curl -sS https://getcomposer.org/installer | php

mv composer.phar /usr/local/bin/composer


Для проверки версии установленного composer надо дать команду:


composer --version


или


php composer --version


Все необходимые пакеты, которые должен установить composer, прописываются в файле composer.json. Для того чтобы создать его, необходимо ввести следующую команду в консоли, находясь внутри папки с вашим проектом:


composer init


Если php-скрипты не выполняются автоматически, то во всех командах слово composer заменяется на php composer. То есть так:


php composer init


В результате работы этой команды будет запущен интерактивный консольный интерфейс. В нем сначала будут запрошены название проекта (сайта), имя автора и прочее. Если не знаете что писать, можно нажимать Enter. Затем запустится диалог поиска пакетов "Search for a package". В нем можно выбрать пакеты, которые надо установить (можно ввести часть имени, например yii). Находите пакет который хотите установить, а потом смотрите необходимую его версию на сайте https://packagist.org.


В принципе, выбранный пакет можно вручную прописать в файл composer.json. После чего дать команду установки всего того, что прописано в этом файле:


composer install


Либо можно воспользоваться командой вида:


composer require --dev phpunit/phpunit ^5.7


Данная команда сначала пропишет указанный пакет phpunit в файл composer.json, а потом его установит.


Все. После этого можете с улыбкой наблюдать как в вашем проекте появилось все что вы хотели скачать.
