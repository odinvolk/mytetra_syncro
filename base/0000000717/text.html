Программы для разбора djvu обычно входят в пакет djvulibre или djvulibre-bin.

ddjvu - Разбирает файлы djvu в файлы других графических форматов. Интерес представляет разбор в TIFF формат. При разборе генерируется многостраничный TIFF файл. Пример команды

ddjvu -format=tiff file.djvu file.tif

Затем TIFF-файл можно разобрать на отдельные картинки через программу tiffsplit (эта программа обычно входит в пакет libtiff-tools).

С помощью ключа -page можно выдергивать только нужные страницы. Формат ключа следующий

-page=1-10 (взять с 1 по 10 страницу)

-page=1,3,5,10-20 (взять 1, 3, 5 страницы и страницы с 10 по 20)
