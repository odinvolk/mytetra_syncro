Для того, чтобы указать порядок записей в запросе, нужно воспользоваться ключевым словом ORDER BY, после которого указывается поле, по которому должна пройти сортировка и направление сортировки.


Направление сортировки может быть двух видов - прямая (ASC) и обратная (DESC). Для прямой сортировки ключевое слово ASC можно не указывать. Для обратной сортировки нужно указывать ключевое слово DESC, вот так:


SELECT * FROM base ORDER BY id DESC
