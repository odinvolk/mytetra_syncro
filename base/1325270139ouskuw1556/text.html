Для сравнения бинарных файлов в Linux можно воспоьлзовться программой vbindiff:


vbindiff file1 file2


Откроется ncurses-интерфейс, в котором интерактивно можно просматривать различия. Выглядит он вот так:



Сверху и снизу показывается один и тот же кусок, взятый из разных файлов. Есть возможность поиска в шестнадцатеричном и текстовом виде. Можно даже редактировать первый (верхний) файл, правда есть какое-то ограничение, когда режим редактирования не включается. Возможно, это зависит от размера файла.
