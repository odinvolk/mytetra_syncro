NConvert

Come to discover NConvert, a batch utility to convert graphic files! Support of more than 400 graphics formats!

The latest version (6.05/3.87) exists for Windows, Linux x86, Linux ppc, FreeBSD x86, OpenBSD x86, NetBSD x86, Solaris sparc, Solaris x86, Irix mips, HP-UX, AIX !

Для коммерческого использования нужно активировать соглашение.

По всей видимости, это сборка UNIX-программы convert.

Ей можно делать многостраничные TIFF файлы из набора отдельных файлов.
