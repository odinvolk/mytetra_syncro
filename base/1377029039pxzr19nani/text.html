На чистом Raspbin не работает зук из коробки. Для того, чтобы звук заработал, следует сделать следующее:


sudo apt-get install alsa-utils

sudo modprobe snd_bcm2835


Загрузку модуля можно добавить в автозагрузку.


Возможно, сработает прописывание строки snd_bcm2835 в файле /etc/modules.


Если такое решение не сработает, надо попробовать прописать модуль в файл /etc/moprobe.d/alsa.conf. Формат данного файла неясен, прописывать по аналогии.
