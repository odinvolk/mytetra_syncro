Чтобы вызвать из одного метода класса другой метод того же класса, нужно пользоваться значением self.

Пример:

class Simple:


def __init__(self):

self.list = []


def f1(self):

self.list.append(123)


def f2(self):

self.f1() # Вызов метода f1() этого же класса

s = Simple()

s.f2()

print s.list


Результат:


[123]
