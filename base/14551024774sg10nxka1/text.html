Русский в zip-архивах


Бывает иногда берешь открываешь архив, а в нем вместо букв сплошные крокозяблы. Что бы исправить баг с неправильным отображением русских названий файлов внутри zip-архивов, делаем следующие. Добавляем два репозитория:


$ nano /etc/apt/sources.list


deb http://ppa.launchpad.net/r0lf/ppa/ubuntu maverick main

deb http://ppa.launchpad.net/frol/zip-i18n/ubuntu maverick main


добавляем ключ для репозитория:


$ sudo add-apt-repository ppa:frol/zip-i18n

$ sudo apt-get update


Устанавливаем пакет libnatspec0:

$ sudo apt-get install libnatspec0


и после ставим пакет p7zip-full.

$ sudo apt-get install p7zip-full


Архивы, открываемые через этот пакет, должны отображаться правильно даже если имена файлов содержат кириллистические символы.
