Как подключить плагин jQuery

Для начинающих разработчиков многие вопросы, порой даже самые простые, кажутся сложными и непонятными. Так и с вопросом «Как подключить плагин jQuery?». В действительности все просто. Опишу это в виде последовательности необходимых действий, а потом покажу пример кода.

  1. Кладем js-файл jQuery в отдельную папку, например /ajax/ и подключаем его через тег SCRIPT.
  2. При необходимости то же самое проделываем с jQuery UI.
  3. Берем js-файл плагина и кладем его в ту же папку. Затем подключаем таким же образом.
  4. Если для работы плагины необходим CSS файл, то то же самое делаем с ним.

Пример кода:

<!-- Подключаем jQuery -->

<script src="/ajax/jquery.js" type="text/javascript"></script>

<!-- При необходимости подключаем jQuery UI -->

<script src="/ajax/jquery.ui.draggable.js" type="text/javascript"></script>

<!-- Подключаем плагин -->

<script src="/ajax/jquery.plugin.js" type="text/javascript"></script>

<!-- При необходимости подключаем CSS необходимое для работы плагина -->

<link href="/ajax/jquery.plugin.css" rel="stylesheet" type="text/css" media="screen" />
