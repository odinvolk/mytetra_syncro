Программа zip не может во время паковки создать архив с разбиением на файлы нужного размера. Поэтому, чтобы получить zip-архив, разбитый на файлы, вначале нужно сделать обычный zip-архив, а затем разбить его с помощь программы zipsplit.

Пример создания zip-архива:

zip -r archivename *

- эта команда упакует текущий каталог со всеми подкаталогами.

Затем надо запустить программу zipsplit

zipslit -n 10000000 archivename

- эта команда разобъет архив на куски по 10Mb. У программы zipsplit есть ограничение - она не может разрезать архив, если в нем лежит хотя бы один файл, размером превышающий размер одного куска.
