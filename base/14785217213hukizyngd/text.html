Перенос доменов .RU, .РФ от АО «Регистратор Р01» к другому Регистратору

Для передачи доменов из АО «Регистратор Р01» Администратору следует:

  1. Удостовериться, что его данные проверены регистратором: в этом случае, в личном кабинете в разделе «Администраторы доменов» в подробной информации по администратору поле «Информация проверена» имеет значение «Да».

    2. Если данные не проверены, то необходимо предоставить копии подтверждающих документов.

    Для физических лиц и индивидуальных предпринимателей:

    • Разворот паспорта с фото (обязательно),
    • Разворот паспорта с отметкой о постоянной регистрации (обязательно),
    • Разворот паспорта с отметкой о ранее выданном документе (если имеется).
  2. Подать заявку на получение кода переноса (AuthInfo-кода)* домена в личном кабинете в разделе «Домены» одним из следующих способов:
    • если нужно получить код переноса для одного домена, необходимо кликнуть по имени домена и в блоке «Инструменты» пройти по ссылке - «Кода переноса»;
    • если нужно получить код переноса для нескольких доменов, выбрать домены из списка, отметив галочкой, затем выбрать для отмеченных доменов групповую операцию «Подать заявку на выдачу кода переноса», нажать «выполнить».

    При оформлении заявки проверить корректность данных Администратора, обязательных для проведения процедуры смены регистратора:

    • адрес электронной почты администратора домена,
    • номер телефона администратора домена.

    По умолчанию эта информация берется из регистрационных данных Администратора домена: «адрес электронной почты» и «номер телефона». Администратор домена может изменить эту информацию до подачи заявки на выдачу кода переноса. Если у Администратора уже есть действующие заявки на получение кода переноса либо выдан действующий код переноса, изменение адреса электронной почты и номера телефона администратора домена, предназначенных для проведения смены регистратора, запрещено.

  3. Выбрать один из способов получения кода переноса:
    • в письменном виде,
    • с использованием авторизации по электронной почте,
    • с использованием sms-авторизации.
  4. Если выбран способ получения кода переноса - в письменном виде, Администратору домена нужно приехать в офис R01 в течение 3 рабочих дней с момента оформления заявки и получить код.

    5. Для получения кода переноса в офисе Администратору домена необходимо иметь при себе следующие документы:

    • физическому лицу и ИП — паспорт,
    • представителю юридического лица:
    • руководителю юридического лица необходимо иметь при себе паспорт;
    • представителю юридического лица, не являющимся руководителем юридического лица, необходимо иметь при себе паспорт и доверенность на право совершать все необходимые действия, связанные с передачей обслуживания доменных имен иному регистратору (Образец доверенности);
    • копию Свидетельства о внесении записи о юридическом лице в Единый государственный реестр юридических лиц (Свидетельство ОГРН).
  5. Если выбран способ получения кода переноса - с использованием авторизации по электронной почте или sms-авторизации, то Администратору домена в зависимости от способа авторизации на адрес электронной почты или на номер телефона, предназначенный для проведения смены регистратора, будет отправлено сообщение, содержащее ключ подтверждения заявки.

    6. Администратору домена нужно подтвердить заявку на получение кода переноса в личном кабинете в разделе «Домены» - «Заявки на выдачу кода переноса для .RU, .РФ доменов»: найти соответствующую заявку и ввести ключ.

    Дополнительно Администратору домена будет отправлено письмо по электронной почте со ссылкой на страницу личного кабинета, на которой можно подтвердить заявку на получение кода переноса или отказаться от нее.

    Внимание! Подтвердить заявку на получение кода переноса для домена нужно в течение 3 рабочих дней.

    После подтверждения заявки будет автоматически сгенерирован код переноса.

  6. Посмотреть код переноса можно в личном кабинете Администратора домена в разделе «Домены»: кликнуть на имя домена, в блоке «Инструменты» выбрать «Код переноса».
  7. Затем обратиться к новому Регистратору с просьбой начать процедуру смены Регистратора в Реестре, сообщив ему код переноса (AuthInfo-код) домена.

Код переноса (AuthInfo-код) используется для защиты от несанкционированной смены регистратора. При смене регистратора AuthInfo-код необходимо сообщить регистратору, которому передается домен. Без кода переноса или с неверным кодом переноса реестр отклонит запрос нового регистратора на перенос домена.

Срок действия AuthInfo-кода — 20 календарных дней.

AuthInfo-код становится недействительным:

  • После передачи поддержки домена другому регистратору
  • После передачи права администрирования домена другому лицу
  • В случае отказа текущего Регистратора в передаче поддержки домена по основаниям, предусмотренным Правилами регистрации доменных имен в доменах .RU и .РФ и Регламентом «О процедуре, подлежащей применению при передаче поддержки сведений о доменном имени между регистраторами».

Условия, при которых смена Регистратора для домена невозможна:

  • истек срок регистрации доменного имени;
  • в отношении доменного имени установлены судебные или досудебные ограничения;
  • если Регистратор R01 запрашивал у Администратора домена дополнительные сведения, и таковые не были предоставлены в установленный срок;
  • прошло менее 30 дней с момента получения Администратором права администрирования от другого лица;
  • прошло менее 30 дней с момента смены Регистратора, осуществляющего поддержку доменного имени.
