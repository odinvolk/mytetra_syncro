Понадобилось запустить оснастку WINS на своей рабочей станции. Да, это не нормально в 2011 году, знаю. Но в Windows RSAT эта оснастка отсутствует. В общем-то она отсутствует там со времён Windows Vista. А дабы не забыть процедуру добавления этой оснастки публикую её здесь.

1. Для начала скачать вот этот архив. Он содержит в себе файлы оснастки из win2003.

2. Скопировать файлы в системную директорию:

Для win x86: Скопировать файлы в %systemroot%\system32

Для win x64: : Скопировать файлы в %systemroot%\sysWOW64

3. Зарегистрировать библиотеку в системе. Для этого запустить cmd.exe с повышенными привилегиями и выполнить:

regsvr32 winssnap.dll

4. Потом запустить скопированный winsmgmt.msc

После этого оснастка будет добавлена в общий список и можно будет использовать её из mmc.
