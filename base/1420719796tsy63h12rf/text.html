Чтобы производить низкосистемные действия с телефоном, нужно активировать режим разработчика.


В телефонах Huawei Honor 6 получение режима разработчика происходит так:


  1. Нужно зайти в меню Настройки ->О телефоне и тапнуть 7-10 раз по Номер сборки, после чего в настройках появится пункт Для разработчиков.
  2. Далее нужно войти в меню Для разработчиков и включить галку Отладка по USB.


После этих действий компьютер должен увидеть смартфон, подключаемый по USB кабелю. Для WIndows (возможно) необходимо будет установить драйвера с официального сайта Huawei. Для Linux никаких драйверов не требуется.
