MRIcroGL

MRIcroGL программа для отображения медицинских 3D изображений. Программа имеет встроенный скриптовый движок для добавления эффектов. Написана на FreePascal/Lazarus и имеет версии для Windows, Linux и MacOSX.

http://mri.aip.is/

RSM3D by Fachhochschule Wiesbaden

RSM3D is a modular software framework for 3d reconstruction from very few X-ray images, which may be virtually freely distributed in 3D space with respect to one another.

http://coin3d.org/usage/casestudies/users/rsm3d

TeleInViVo

TeleInViVo is a collaborative volume visualization application developed at Fraunhofer CRCG.
