Иногда возникает необходимость настроить два IP адреса на один физический интерфейс. Это можно сделать следующим образом, согласно Debian Wiki.


Надо открыть файл /etc/network/interfaces, и привести его к нужному виду. Ниже показан пример, в котором одному интерфейсу eth0 назначается три IP адреса в основной сети, и один IP адрес в другой подсети:


auto eth0
allow-hotplug eth0
iface eth0 inet static
    address 192.168.1.42
    netmask 255.255.255.0
    gateway 192.168.1.1

# Дополнительные IP адреса в основной сети

auto eth0:0
iface eth0:0 inet static
    address 192.168.1.43
    netmask 255.255.255.0

auto eth0:1
iface eth0:1 inet static
    address 192.168.1.44
    netmask 255.255.255.0

# также возможно добавить IP адрес из другой подсети

auto eth0:2
iface eth0:2 inet static
    address 10.10.10.14
    netmask 255.255.255.0


Нужно отметить, что опция gateway прописывается только для одного интерфейса. Ее нельзя прописывать дважды.


Далее сетевая подсистема перегружается:


# /etc/init.d/networking restart


После чего можно проверить наличие нескольких адресов через команду:


# ip addr
