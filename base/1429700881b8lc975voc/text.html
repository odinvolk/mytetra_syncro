Есть некий командлет, например Test-Path. Нужно получить всю возможную информацию о нем.


Можно выполнить следующую команду:


PS C:\Windows\system32> Get-Command Test-Path


CommandType Name Definition

----------- ---- ----------

Cmdlet Test-Path Test-Path [-Path] <String[]> [-Filter <String>] ...


Это очень краткая информация, и по ней невозможно определить, например, в какой модуль входит коммандлет.


Для получения более подробной информации, надо "развернуть" все свойства, полученные в объекте, выданном коммандлетом Get-Command. Это можно сделать с помощью коммандлета Format-List (сокращенно fl), который форматирует вывод как список свойств, в котором каждое свойство выводится на новой строке:


PS C:\Windows\system32> Get-Command Test-Path | fl *
