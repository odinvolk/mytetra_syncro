SI_Browser 1.2 - просмотрщик картинок для врачей

Программа позволяет просматривать картинки в формате DICOM (основной стандарт для пересылки медицинской информации). Написана на FreePascal/Lazarus с использованием GTK2.

Новые функции:

  • новый интерфейс пользователя.
  • добавлен формат JPEG2000.
  • добавлено управление контрастом и яркостью.
  • добавлены элементы управление для просмотра информации DICOM через оверлей.

Программа бесплатна и доступна для скачивания после регистрации.
