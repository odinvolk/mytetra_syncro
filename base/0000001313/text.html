Скрипт gnuplot должен начинаться со стоки

#! /usr/bin/gnuplot -persist

Далее устанавливаются параметры терминала, в которых почему-то указываются и параметры вывода

Для вывода в формате eps (файл *.ps) следует использовать такую строку

set terminal postscript eps enhanced color solid

Для вывода в формате PNG следует использовать такую строку

set terminal png small

Выходной файл указывается командой

set out "sine.png"
