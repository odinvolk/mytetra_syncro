Утилита fakeroot перехватывает системные вызовы chmod(2) и stat(2) для файлов, и возвращает значения, как если бы файл принадлежал пользователю root. Небольшой пример:

$ id

uid=1000(bvk) gid=1000(bvk) ...

$ touch trololo

$ ls -l trololo

-rw-r--r-- 1 bvk bvk 0 2010-10-22 13:23 trololo

$ fakeroot ls -l trololo

-rw-r--r-- 1 root root 0 2010-10-22 13:23 trololo
