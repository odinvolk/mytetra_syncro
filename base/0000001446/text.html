Guitarix


В debian надо собирать из исходников. Сборка нестандартная:


./waf configure

./waf build

sudo ./waf install


В процессе конфигурации будут ошибки, надо доустанавливать dev-пакеты, такие как:


libjack-dev

libjack0.100.0-dev

libjackasyn-dev

libflac-dev

libsndfile1-dev

ladspa-sdk


все эти пакеты есть в Debian Lenny 5.0.4.
