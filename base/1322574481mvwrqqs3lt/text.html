В чужих проектах бывает так, что есть начальный "зазор" по времени, то есть самая первая задача начинается позже старта проекта.


Чтобы это исправить, надо посмотреть дату начала первой задачи в проекте, и указать ее в окне:


Проект -> Сведения о проекте -> Дата начала


Затем надо активировать панель инструментов "Анализ" (по правой кнопке мыши в области панелей инстументов). На панели "Анализ" есть кнопка "Корректировка дат". При ее нажатии надо ввести ту же дату начала проекта, которую вводили на предыдущем шаге.
