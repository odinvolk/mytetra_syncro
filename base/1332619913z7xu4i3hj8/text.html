Повторяю - На официальном сайте CASIO - программа IDES 3.0 для старых типов синтезаторов (файл WK3-30.exe). При инсталляции создает каталоги:


c:\Program Files\CASIOWK-3700


в нем 3 подкаталога:


converter

Data Manager

Parameter Editor


Работаешь с converter\rhythmconv.exe - это и есть Rhythm Converter.


Если непонятно, читай нашу ветку по СТК 5000.



Дополнительная информация


Существуют следующие версии IDES:


  • IDES 2.0 for the models CTK-691, WK-3000 and WK-3500
  • IDES 3.0 for the models CTK-900, WK-3200 and WK-3700
  • IDES 4.0 for the models PX-410R, WK-3300, WK-3800 and WK-8000


Страница для сачивания: http://www.casio-europe.com/euro/emi/specials/ides/downloads/
