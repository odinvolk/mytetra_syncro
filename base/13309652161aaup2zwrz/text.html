Обычно, MKV-файлы используюся в качестве сырого материала, и объединять их нет никакого смысла. Поэтому здесь показан пример команд, которые перегоняют каждый MTS-файл в файл формата MKV (Matroska). А затем с помощью программы mkvmerge происходит генерация одного MKV-файла из двух других MKV-файлов:


ffmpeg -i 1.MTS -acodec copy -vcodec copy -sn 1.mkv

ffmpeg -i 2.MTS -acodec copy -vcodec copy -sn 2.mkv

mkvmerge -o 12.mkv 1.mkv + 2.mkv
