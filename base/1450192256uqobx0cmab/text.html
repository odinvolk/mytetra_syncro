Сервис пополнения и вывода WMX


Сервис пополнения и вывода титульных знаков WMX обеспечивает хранение и передачу прав на BTC - электронные денежные единицы системы Bitcoin, а также учет их в титульных знаках WMX по курсу 1 WMX = 0.001 BTC.

Гарантом титульных знаков WMX выступает компания INDX Transactions LTD, условия предоставления услуг которой изложены в оферте (Договоре хранения), акцептуемом участниками системы при регистрации кошелька типа X. Права владельца переданных на хранение BTC подтверждаются зачислением на кошельки типа Х соответствующего объема складских расписок - WMX.

Зачисление WMX на X-кошелек производится Гарантом только после получения не менее шести подтверждений в базе данных пиринговой сети bitcoin.org операции зачисления BTC на предоставленный Гарантом bitcoin-адрес.

На хранение Гарантом принимаются любые переводимые в Bitcoin BTC, однако зачисление WMX происходит только для значений превышающих 0.0001 BTC.

Возврат с хранения BTC возможен только для объемов превышающих эквивалент 1.00 WMX. При возврате с хранения Гарантом удерживается фиксированная комиссия за операцию в размере 0.0005 BTC, которая уплачивается при осуществлении транзакции.

Адрес сервиса: wmx.wmtransfer.com
