Для этого можно воспользоваться командой:

# LANG=ru_RU.UTF-8 xterm -e ssh user@hostname

В этом случае будет открываться окно с маленьким шрифтом. Русские символы будут видны, но с мелким шрифтом неудобно работать.

Чтобы открыть окно с большим шрифтом, надо убедиться, что есть файл ~.Xdefaults. Если его нет, его можно создать. Содержимое этого файла должно быть таким:

*VT100.utf8Fonts.font: -misc-fixed-medium-r-normal--20-200-75-75-c-100-iso10646-1

*VT100*font: -misc-fixed-medium-r-normal--20-200-75-75-c-100-iso10646-1

Да, в обоих строках кодировка именно iso10646-1. Вместо "20" можно писать другой размер шрифта. Более тонкий и красивый шрифт дадут такие настройки:

*VT100.utf8Fonts.font: -*-fixed-medium-*-*--18-120-*-*-*-*-iso10646-1

*VT100*font: -*-fixed-medium-*-*--18-120-*-*-*-*-iso10646-1

Еще желательно добавить такие опции:

XTerm*foreground: grey

XTerm*background: black

XTerm*scrollBar: true

XTerm*saveLines: 1000

После модификации файла ~.Xdefaults надо дать команду:

# xrdb .Xdefaults

В ответ может вылезти ошибка:

<stdin>:0: warning: Unknown encoding: ru_RU.KOI8-R

но ее можно игнорировать, всеравно все будет работать.

Далее надо запустить всю ту же первую команду, и будет открываться окно с большим шрифтом.
