Mp3 Audio Editor


Mp3 Audio Editor — программа для редактирования mp3-файлов. Это один из немногих редакторов, которые умеют редактировать MP3 без распаковки и перекодировки.


Программа умеет напрямую работать с MP3: разрезать, склеивать, вырезать куски, вставлять, микшировать несколько файлов. Помимо редактирования, возможна запись в WAV, MP3, WMA, OGG с любого внешнего источника звука. Для художественной обработки звука можно воспользоваться большой библиотекой эффектов (Amplify, Compressor, Delay, Equalize, Fade In and Fade Out, Flanger, Invert, Normalize, Phaser, Reverb, Reverse, Silence, Shrink, Stretch, Vibrato). Свои творения можно записать на аудио CD прямо из MP3, WMA, WAV и OGG. Чтобы пользоваться программой больше триального периода, нужно прикупить лицензионный ключик – от 30 до 50 долларов, в зависимости от способа использования (для бизнеса или для дома). Mp3 Audio Editor работает в Windows 98/Me/2000/XP/Vista/7. Если бы не цена, то можно сказать, что программа мне понравилась – красивая, понятная, удобная.


Сайт - http://www.mp3audioeditor.com/



mp3DirectCut


mp3DirectCut - это быстрый аудиоредактор и рекордер с широкими возможностями для работы со звуком, сжатым в mp3. Вы можете напрямую вырезать, копировать, вставлять или изменять громкость без необходимости перекодировать MP3 файлы для редактирования аудио. Это экономит время кодирования и сохраняет исходное качество звука, потому что ничего не будет перекодировано. Встроенный рекордер создает mp3 на лету прямо с аудиовхода. Используя Cue-листы, функцию обнаружения паузы (Auto cue), можно легко разделить длинные файлы.


Программа бесплатная, работает даже в Linux под Wine. Сама программа легковесная, никаких дополнительных кодеков не требуется.


Сайт - http://mpesch3.de1.cc/mp3dc.html
