Для перекодировки из формата AC3 в формат WAV, можно воспользоваться командой:

# ffmpeg -i file.ac3 file.wav

При переконвертации из многодорожечного AC3 файла, создается WAV-файл с таким же количеством дорожек.

Внимание! Полученный файл может быть некорректным, например, содержать неправильный заголовок, в котором прописано неправильная длительность аудио-содержимого.

В этом случае, надо переконвертировать программой transcode.
