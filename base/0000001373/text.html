На 2010 год имеются следующие наименования дистрибутивов Debian

Etch (Debian 4.0) - старая стабильная ветка

Lenny (Debian 5.0) - Stable, текущая стабильная ветка

Squeeze (Debian 6.0) - Testing, тестируемая ветка, она затем переименовывается в стабильную

Sid - текущая нестабильная (unstable) ветка, номера версии у нее нет

Testing - текущая тестируемая (testing) ветка, номера версии у нее нет. Более стабильна чем unstable.

Есть еще репозитарий бэкпортов, в котором есть бэкпорты для стабильной ветки. Для Etch ссылка на него выглядит так:

deb http://www.backports.org/debian/ etch-backports main contrib non-free

Для Lenny, видимо, будет выглядеть так:

deb http://www.backports.org/debian/ lenny-backports main contrib non-free
