Проверка обновлений:

yum check-update

Обновление одного пакета. Для обновления можно выбрать одиночный пакет, множество пакетов или же все пакеты сразу. Если какие-либо зависимости пакетов, выбранных для обновления, сами имеют более свежие версии, они также будут обновлены. Для обновления одного пакета можно использовать команду:

yum update имя_пакета

Поиск пакетов, содержащих нужную строку в названии или описании пакета:

yum search часть_имени_пакета

Поиск пакетов, содержащих файл, происходит так:

yum provides имя_файла

Вывести список всех пакетов, установленных в систему:

yum list

Узнать номер версии пакета можно так:

# yum list | grep postgre

postgresql-libs.i386 8.1.18-2.el5_4.1 installed

freeradius-postgresql.i386 1.1.3-1.6.el5 base

freeradius2-postgresql.i386 2.1.12-4.el5_8 updates

Удалить пакет:

yum remove package_name
