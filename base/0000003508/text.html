Возможные команды:

git pull

- получить из репозитария все последние изменения.

git add .

- поместит в stage все измененные файлы, добавит в stage все новые файлы.

git commit -a -m "commit comment"

- совершит коммит, автоматически индексируя изменения в файлах проекта. Новые файлы этой командой индексироваться не будут! (Но они будут уже проиндексированы командой "git add ."). Удаление же файлов будет учтено.

git push

- отправить свои изменения в удаленную ветку, созданную при клонировании по умолчанию.
