Иногда при работе с базами данных появляется необходимость замены всего текста или части текста на другой. Проделать это можно с помощью следующего запроса:


UPDATE table SET field = REPLACE(field, 'Текст 1', 'Текст 2');

где:


  • table - название таблицы,
  • field - название поля, содержимое которого нужно изменить,
  • Текст 1 - текст, который нужно заменить,
  • Текст 2 - текст, на который нужно заменить.


Примечание: Данный совет и синтаксис подходит как для SQLite, так и для MySQL.


Реальный пример:


UPDATE automobile_models SET model_name = REPLACE(model_name, 'Touring', 'Универсал');


Заменяет подстроку 'Touring' в названии модели, на 'Универсал', остальной текст останется не тронутым.
