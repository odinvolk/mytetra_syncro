Для монтирования втсроенного в Dingoo диска можно воспользоваться командой вида:

sudo mount -t vfat /dev/sdc /media/usb

Вставленная в Dingoo флешка монтируется автоматом. Команда mount показывает следующее:

/dev/sdd1 on /media/disk type vfat (rw,nosuid,nodev,uhelper=hal,uid=500,utf8,shortname=mixed,flush)
