Как включить отладку по USB в планшете Samsung SM-T3110 Galaxy Tab 3 8.0? Планшет Samsung SM-T3110 Galaxy Tab 3 8.0


ИСТОЧНИК: 4PDA.ru

Как включить отладку по USB?


  1. Нажать Настройки> Об устройстве> тапнуть 8 раз подряд по пункту номер сборки.
  2. Появиться меню для разработчиков.
  3. В этом меню выбираем пункт "Отладка по USB", ставим галку напротив.


Я проверила, действует!
