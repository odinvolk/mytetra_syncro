Вопрос

В журнале документов в форме списка есть реквизит Список.

В нем указано опция "Произвольный запрос ".

И есть запрос который строит список документов. Как в этот запрос передать значение переменной &ПользовательИБ ???

В самом запросе планирую использовать конструкцию с амперсандом &:

... ГДЕ

ЖурналДокументовЕжедневныеОтчетыСотрдников.Сотрудник.ПользовательИБ = &ПользовательИБ

Ответ:

У формы, на которой лежит список, нужно открыть обработчик:

ПриСозданииНаСервере()

И вписать следующую команду:

Список.Параметры.УстановитьЗначениеПараметра("ИмяПараметра", ЗначениеПараметра);

Список - это реквизит формы т.е. динамический список.
