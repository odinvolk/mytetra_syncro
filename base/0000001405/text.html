Если при запуске CBuilderX появляется ошибка:

No protocol specified

то скорее всего запуск осуществляется из-под root, а система не дает пользователю root подключаться к X-серверу из соображений безопасности.

Чтобы обойти это ограничение, надо в файл ~/.profile текущего пользователя добавить команду:

xhost +local:root

После чего надо перелогиниться, и снова попытаться запустить CBuilderX из-под root с помощью su или sudo. Программа должна нормально запуститься.
