Для безопасного получения информации из ассоциативных массивов, например из массива $_POST, можно воспользоваться классом Arr.

Пример получения данных:

$login=Arr::get($_POST, 'login', '');

$password=Arr::get($_POST, 'password', '');

Как видно из кода, используется статический метод get() класса Arr.

Первый аргумент - ассоциативный массив, из которого надо извлечь данные, второй аргумент - ключ (имя) извлекаемого значения, третий аргумент - значение по-умолчанию. Третий аргумент можно не указывать, но лучше указывать для предсказуемости поведения программы.
