Alfresco — Оpen Source система управления контентом уровня предприятия, с открытыми стандартами, для платформ Microsoft Windows и Unix.


Система характеризуется дизайном с высокой степенью модульности и масштабируемой производительностью.


Alfresco включает в себя такие модули как репозиторий контента, готовый к использованию веб-портал, фреймворк для управления и использования стандартным содержимым портала, интерфейс CIFS, предоставляющий совместимость с файловыми системами Microsoft Windows и Unix ОС через интернет, а также предлагает механизм индексирования и поиска Lucene и jBPM workflow.


Система Alfresco разработана с использованием Java технологий.


Alfresco ECM – свободная (open source) система для организации корпоративного документооборота, интранет-портала и электронного архива. Благодаря простоте настройки и надежности в работе Alfresco ECM уже используется в сотнях компаний по всему миру.



Есть ли еще бесплатные системы электронного документооборота?


Есть, например Наудок, указанный ранее. Также есть ряд разработчиков которые предлагают свои моно-пользовательские системы бесплатно, а также есть разработчики, которые предлагают версии для малых фирм (около 5 пользователей), с поддержкой на форуме, на довольно длительный триальный период, например fossdoc.


Посмотрите систему OpenGoo (http://www.opengoo.org) - очень приятный русскоязычный интерфейс и приличная функциональность. Система открытая и доступна для скачивания, регулярно обновляется.
