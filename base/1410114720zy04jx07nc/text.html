Подробный отчет о сайте можно получить через сервис xtool.ru


Пример:


http://xtool.ru/trast.php?site=webhamster.ru


При анализе сайта выдается следующая информация:


  • Ссылок внешних\внутренних
  • Оценка трафика
  • Внешние ссылки на сайт
  • Заспамленность ссылками
  • Санкции в Яндексе, АГС
  • Данные Solomono
  • Ссылки на сайт (количество, с какого уровня страниц)
  • Ссылки с сайта (количество, с какого уровня страниц)
  • Количество анкоров на сайт
  • Количество анкоров с сайта
  • Раскрученность в Yandex ВЧ/СЧ/НЧ
  • Раскрученность в Google ВЧ/СЧ/НЧ
  • Возраст сайта, дней (лет)
  • Индексация в Яндексе (страниц)
  • Скорость обновления по Яндексу (страниц/сутки)
  • Тиц и PR


Далее графики:


  • График изменения траста XT для сайта
  • История изменения ТИЦ
  • История изменения PR
  • История индексации в Яндексе
  • История посещаемости по данным LiveInternet
  • Поисковый траффик Google
  • Информация Whois


И в конце есть аналитическая таблица "Видимость сайта в поисковых системах по ключевым словам" (это и есть семантическое ядро сайта):


  • Общая видимость
  • Видимость ключевых слов сайта в Яндекс
  • Видимость ключевых слов сайта в Google
  • Видимость ключевых слов сайта в Backlinks
