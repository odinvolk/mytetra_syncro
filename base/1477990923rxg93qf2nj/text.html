Комментарии в SQL обозначаются символами --. Комментарии можно размещать даже между частями SQL-запроса.


Вот пример:


CREATE TABLE ideas(
-- поле с автоматически увеличивающимся счётчиком
idea_id INTEGER PRIMARY KEY,
-- детали идеи
idea_body TEXT,
-- как хорошо она работает
success_rating INTEGER
);
