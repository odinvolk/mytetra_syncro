Если появляется проблемма с ошибкой:


Warning: include(ci_application/errors/error_php.php): failed to open stream: No such file or directory in /media/mx_120_part_2/home/xi/work/game_ksi/webwork/linuxtrash/ci_system/core/Exceptions.php on line 182


То это значит, что вы наткнулись на давний застаревший баг в CodeIgniter 2.x.x.


Решение:


Exceptions.php line 182 is the default error handler but the path does not provide the full path.
So when we have an error from anywhere from outside, the system could not find the file to include.


В файле Exceptions.php надо внести исправление:


Original

include(APPPATH.'errors/error_php.php');

Fix

include(FCPATH.APPPATH.'errors/error_php.php');


Решение справедливо для CI 2.1.0+ and 3.0dev


Внимание! Данный инклюд в файле встречается не в одном месте! Исправлять нужно все места.
