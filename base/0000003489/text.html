Менеджер загрузки GRUB2 инсталлируется в MBR указанного диска с помощью команды

# grub-install /dev/sda

Если на этапе установки возникла ошибка, можно воспользоваться командой

# grub-install /dev/sda --recheck

или

# grub-install /dev/sda --recheck --root-directory=/media/dir

где --root-directory - это корень примонтированного диска, на котором должны разместиться рабочие файлы GRUB2.

Установка из консоли grub2 с помощью команды setup в grub2 не поддерживается, как это было в grub первых версий.
