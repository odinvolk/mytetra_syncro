Для преобразования QString в строку типа string стандартной библиотеки, можно воспользоваться такими командами:



QString qs;


// Для данных в UTF-8 формате

std::string utf8_text = qs.toUtf8().constData();


// Или с преобразованием в 8-бит формат, например для Wundows

std::string current_locale_text = qs.toLocal8Bit().constData();
