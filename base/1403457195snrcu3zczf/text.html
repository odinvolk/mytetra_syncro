Несмотря на приняетые в CodeIgniter соглашения о кодировании, в данном фреймверке можно использовать верблюжий регистр вИменахОбъектах.

Для контроллеров и видов имена в верблюжьем регистре поддерживаются из коробки. Но верблюжий регистр не работает для моделей.

Чтобы исправить, можно сделать следующее:

Открыть файл /ci_system/core/Loader.php, и заменить в нем все использования функции strtolower() на lcfirst().

По- хорошему, это нужно оформить в виде хука ядра.
