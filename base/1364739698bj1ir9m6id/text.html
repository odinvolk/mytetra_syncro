Пример для темы Synthetica


Нужно взять два jar файла


Сам класс Synthetica и оформление Synthetica BlackEye Look and Feel с сайта


http://www.jyloo.com/synthetica/themes/


Jar-файлы надо скопировать в какую-нибудь директорию, например /opt/NetBeans_7_3/netbeans-7.3/bin/netbeans.


Запускать NetBean командой:


/opt/NetBeans_7_3/netbeans-7.3/bin/netbeans

--cp:p /opt/NetBeans_7_3/theme/syntheticaBlackEye.jar

--cp:p /opt/NetBeans_7_3/theme/synthetica.jar

--laf de.javasoft.plaf.synthetica.SyntheticaBlackEyeLookAndFeel


Пример для темы Substance.


Вначале надо собрать Substance. После сборки нужно найти два файла - substance.jar и trident.jar и скопировать их в какую-нибудь директорию NetBeans. Запуск происходит командой:


/opt/NetBeans_7_3/netbeans-7.3/bin/netbeans

--cp:p /opt/NetBeans_7_3/theme/substance.jar

--cp:p /opt/NetBeans_7_3/theme/trident.jar

--laf org.pushingpixels.substance.api.skin.SubstanceBusinessLookAndFeel
