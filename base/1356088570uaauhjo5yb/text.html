1) Доустанавливаем необходимые пакеты

apt-get install libxml2-dev pkg-config curl libcurl-dev libcurl4-openssl-dev libfuse-dev


2) Устанавливаем cloudfuse


mkdir /usr/src/cloudfuse

cd /usr/src/cloudfuse

git clone https://github.com/redbo/cloudfuse.git

./configure

make

make install


3) Создаем файл конфигурации


touch /root/.cloudfuse

nano /root/.cloudfuse

> username=system:user

> api_key=password

> use_snet=false

> auth_url=https://yourserver/auth/v1.0


Все эти параметры можно также указать в командной строке при монтировании.


4) Используем


# cloudfuse /mnt/storage

# df -h /mnt/storage

Filesystem 1K-blocks Used Available Use% Mounted on


cloudfuse 4294967294 0 4294967294 0% /mnt/storage
