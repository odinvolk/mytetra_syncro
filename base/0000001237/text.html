Если постянный запуск Apache или MySql не требуется, автозапуск можно отключить в управлении сервисами.

Чтобы запускать Apache или MySql по необходимости, можно воспользоваться следующими командами (команды отдаются от рута).

Старт Apache:

# sudo /etc/init.d/apache2 start

Останов Apache:

# sudo /etc/init.d/apache2 stop

Старт MySql:

# sudo /etc/init.d/mysql start

Останов MySql:

# sudo /etc/init.d/mysql start
