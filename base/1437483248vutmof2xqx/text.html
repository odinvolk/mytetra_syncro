TopSape (http://topsape.ru) - это рейтинг вебмастеров, работающих с биржей ссылок SAPE, а так же сообщество людей, зарабатывающих в интернете. Так-как у самой биржы ссылок САПЕ (в отличии некоторых других партнерок) нет официального топа, был создан этот, с помощью API Sape (что гарантирует достоверность данных). Максимальный зарегистрированный заработок - 137 тысяч рублей в день. Для регистрации нужно указать логин и md5-хеш пароля от аккаунта.

Рейтинг обновляется автоматически, раз в сутки, в 6-8 часов утра. Данные по сайтам обновляются раз в неделю. Вы можете сменить отображаемое имя/урл в разделе управление. Данные заработка можно использовать у себя на сайте через счетчики или через API.
