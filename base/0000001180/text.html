Есть у меня каталог /media/net, в который через smbmount примонтирован расшаренный каталог на Windows-компе. Windows-комп вырубился. В результате, при попытке получить список файлов в каталоге /media через ls, комп задумывается на минуту, потом показывает такое

# ls -l

dr-xr-xr-x 1 root root 2048 2009-01-01 19:28 cdrom0

?--------- ? ? ? ? ? net

dr-xr-x--- 1 root plugdev 8192 2009-02-19 08:45 sda1

drwxr-xr-x 20 root root 4096 2007-09-13 22:46 sda2

Я удалил все smb-процессы, в результате остался только один загадочный процесс [smbiod]

# ps aux | grep smb

root 8983 0.1 0.0 0 0 ? S< 11:27 0:10 [smbiod]

который через kill -9 не удаляется.

Удалить каталог /media/net невозможно

# rmdir /media/net

rmdir: /media/net: Device or resource busy

Вопрос: как разблокировать каталог /media/net не перегружая комп?

Ответ:

umount -f -l /media/net
