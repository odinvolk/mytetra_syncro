С перекодированием в цветовое пространство 420p:

mencoder -ovc lavc -lavcopts \

vcodec=ffvhuff:vstrict=-1:pred=2:context=1:format=422p video-before.avi \

-o video-compressed-ffvhuff.avi

Без перекодирования:

mencoder -ovc lavc -lavcopts \

vcodec=ffvhuff:vstrict=-1:pred=2:context=1 video-before.avi -o \

video-compressed-ffvhuff-420.avi
