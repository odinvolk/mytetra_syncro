Информация для пользователей Linux


С помощью утилиты convert, входящей в состав пакета ImageMagick, можно создавать PDF-файлы из набора изображений.


Команда очень проста:


convert *.jpg output.pdf


Предватительно файлам нужно задать названия с учетом сортировки, например 001.jpg, 002.jpg ... 00N.jpg. Именно в этой последовательности будут созданы листы PDF-документа.



Информация для пользователей Windows


Программа ImageMagick существует и для этой платформы. При установке инсталлятор просто спросит, в какой каталог установить программу. После установки надо прописать в системную переменну PATH путь к каталогу, где лежит исполняемый файл convert.exe. Помимо convert.exe в том же каталоге будет куча утилит для работы с изображениями, которые входят в пакет ImageMagick:


  • dcraw.exe
  • convert.exe
  • display.exe
  • stream.exe
  • conjure.exe
  • animate.exe
  • hp2xx.exe
  • identify.exe
  • composite.exe
  • montage.exe
  • mogrify.exe
  • imdisplay.exe
  • import.exe
  • compare.exe
  • ffmpeg.exe
