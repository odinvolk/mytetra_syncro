Чтобы сделать функцию, которая будет видна в списке макросов, надо сделать следующее.


1. Добавить в консоли VBA модуль с произвольным именем


2. В модуле сделать процедуру:


Sub Перенумерация_документов()


...


MsgBox "Документы перенумерованы"

End Sub


3. После этого действия, при нажатии на кнопку вызова макросов, в списке макросов будет строка "Перенумерация документов".
