Для просмотра переменных окружения можно воспользоваться командой:


export


Или можно получить переменные в более чистом виде с помощью команды env:


env


Для установки переменных окружения можно воспользоваться следующим синтаксисом команды:


export VARNAME=VarValue


Для удаления переменной окружения используется команда unset.


unset VARNAME


Добавить в переменную PATH дополнительный каталог можно так:


export PATH=$PATH:/usr/share/smartgit/bin
