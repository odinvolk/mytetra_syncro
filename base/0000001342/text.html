Утилита sysv-rc-conf устанавливается с помощью команды:

# apt-get install sysv-rc-conf

Запускается с помощью команды:

# sudo sysv-rc-conf

Представляет собой простой текстовый интерфейс для управления уровнем запуска демонов, и также позволяет запустить/остановить нужный демон.

Недостатки:

1. Показывает первые 10 символов названия демона, поэтому бывает непонятно к какому процессу относится демон

2. Не показывает, запущен ли в данный момент процесс или нет.

3. Все изменения применяет сразу, не спрашивая подтверждения, поэтому пользоваться ей надо аккуратно.
