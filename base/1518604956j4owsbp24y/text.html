1.


Поле "Inherits:" в заголовке описания класса обозначает "Унаследован от:" или, другими словами, "Является наследником класса:".


Пример.


Класс QStringList - это наследник класса QList. Поэтому в документации будет написано:


QStringList Class

...

Inherits: QList



2.


Поле "Inherited By:" в заголовке класса обозначает "Наследниками являются:", после чего перечислены все производные классы от данного класса.


Пример.


Класс QList имеет несколько производных классов. В друментации они будут перечислены таким образом:


QList Class

...

Inherited By: QByteArrayList, QItemSelection, QQueue, and QStringList
