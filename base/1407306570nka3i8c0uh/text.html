Пример добавиления нового репозитария в YUM.

Задача: подключить репозитарий DAG от RedHat6 в Scientific Linux 6.5.

Вначале нужно создать новую секцию в файле /etc/yum.conf.

[dag]
name=Dag RPM Repository for Fedora Core
baseurl=http://apt.sw.be/redhat/el6/en/i386/dag

gpgcheck=1
enabled=1

Лалее надо добавить PGP ключ этого репозитария.

rpm --import http://dag.wieers.com/packages/RPM-GPG-KEY.dag.txt

Далее обновляется кеш YUM:

yum update

И можно инсталлировать программы:

yum install rar
