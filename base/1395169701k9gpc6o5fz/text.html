Предположим, есть картинка, отпределенная в HTML коде так:


<img src="pathTo/myImage.jpg"/>


Задача: требуется убрать из HTML-кода ссылку на картинку, чтобы верстка была полностью семантической.


Для этого можно в HTML-коде прописать:

<img class="myClass" />

И в CSS-файле прописать


.myClass{

content:url("pathTo/myImage.jpg");

}


Такое решение на 18.03.2014 работает в:

  • Chrome 14.0.835.163
  • Safari 4.0.5
  • Opera 10.6

Не работает в:

  • FireFox 27.0
  • IE 11.0
