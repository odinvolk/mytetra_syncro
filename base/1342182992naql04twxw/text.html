На 2012 год найдены следующие проекты:

IE7 Pro

Единственный "рабочий" проект. Вместе с собой ставит два жутких тулбара, если от них отказываться, не будет работать и сам IE7Pro.

Основная проблема - не позволяет создавать свои скрипты. Можно только получать скрипты с сайта iescripts.org. Сайт не обновлялся с 2010 года.

Greasemonkey for IE ( GM4IE )

Мифический проект, о нем есть несколько упоминаний в сети. Однако официальный сайт www.gm4ie.com (по некоторым источникам www.gmie.com) представляет собой киберсквоттеровскую заглушку. На варезных сайтах предлагают скачать какой-то gm4ie, но ставить такое как-то боязно.
