Возможность установки брекпоинта на изменение переменной в разных IDE называетс япо-разному:


  • Visual Studio - data breakpoint
  • Idea - field watchpoint
  • Netbeans - variable breakpoint
  • QtCreator - http://doc.qt.io/qtcreator/creator-debug-mode.html

Подробнее, как это делается и QtCreator: при отладке, в области точек останова надо вызвать контекстное меню по правой кнопке мыли и выбрать Add Breakpoint. Далее надо выбрать тип точки останова:



Далее, скорее всего, надо выбрать Break on memory address. Где и как устанавливать имя переменной - непонятно. Надо разбираться дальше.
