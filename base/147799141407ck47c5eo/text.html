Вопрос:


Есть ли в SQL аналог echo? Когда пишутся длинные файлы с SQL-командами, очень хочется выводить хоть какие-то сообщения. Ну, хотя бы чтобы предварить какой-нибудь запрос поясняющим сообщением. Есть ли в SQL (интересует SQLite3) команда печати в консоль произвольных сообщений?



Ответ:


Можно воспользоваться командой:


select "Произвольное сообщение";


Или, в некоторых диалектах SQL:


SELECT "Произвольное сообщение" AS "";
