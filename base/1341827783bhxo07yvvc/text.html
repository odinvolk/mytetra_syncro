На практике случается так, что нужно запустить программу 1с по ярлыку, без ввода пароля и выбора пользователя. Для того, чтобы реализовать эту задачу нужно не так много, а именно создать ярлык и задать параметры запуска ярлыка 1с.


Для программирования запуска достаточно в ярлыке прописать код:


Для серверного варианта


[путь к файлу 1cv8.exe] enterprise /S[сервер_1с]\Имя_Базы_1с /N [пользователь] /P [пароль]


Для файлового варианта


[путь к файлу 1cv8.exe] enterprise /F[путь к базе] /N [пользователь] /P [пароль]


Также стоит обратить внимание и на имя пользователя: «Если имя пользователя состоит из двух и более слов разделенных пробелом, то нужно его заключить в кавычки, т.к. неправильно будет прочитана строка с именем пользователя. А именно будет прочитано только слово до первого пробела»


Используя такой метод запуска 1с стоит помнить, что ярлык 1с для любого пользователя является открытой книжкой, и через него легко можно ознакомиться с паролем пользователя базы!
