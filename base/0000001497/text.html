Коллекция словарей находится по адресу

http://xdxf.revdanica.com/down/index.php

В верхнем правом углу страницы находится выбор формата словаря. По-умолчанию выбран формат "XDXF", который не понимается ни StarDict (хотя, вроде есть плагин), ни GoldenDict. Нужно выбрать формат "Stardict", нажать "Submit", и будет выдан список словарей в фармате, пригодном для использования в StarDict/GoldenDict.

Прямая ссылка на полный англо-русский словарь:

http://ignum.dl.sourceforge.net/project/xdxf/dicts-stardict-form-xdxf/002c/stardict-comn_sdict05_eng_rus_full-2.4.2.tar.bz2
