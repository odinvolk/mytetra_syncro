В Ubuntu 9.04 отключили перезапуск иксов с помощью Ctrl+Alt+BackSpace, но это можно исправить.

Эта команда управляет включением ctrl-alt-backspace хоткея в иксах.

# dontzap

The only accepted parameters are:

-e, --enable Set «DontZap» to «True» (Ctrl+Alt+Backspace does not restart the xserver).

-d, --disable Set «DontZap» to «False» (Ctrl+Alt+Backspace restarts the xserver).

-h, --help Show the help page.

Как работа этих трех кнопок была отключена?

В /etc/X11/xorg.conf

Section “ServerFlags”

Option “DontZap” “yes”

EndSection

Вот такая строчка вписывается в конфигурационный файл иксов и трипедали перестают работать.

Пользователи, которым нужна данная функциональность, могут включить её через файл конфигурации xorg.conf или воспользовавшись командой dontzap --disable.
