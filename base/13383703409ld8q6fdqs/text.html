Файл *.dt - это файл выгрузки информационной базы.

В нем лежит код, структура данных и данные.

Чтобы сделать *.dt файл, нужно нажать:

Администрирование -> Выгрузить информационную базу

Файл *.cf - это файл конфигурации.

В нем лежит код и структура данных, а самих данных нет.

Следует помнить, что в него не выгружаются пользователи, заведенные в системе.

Чтобы сделать *.cf файл, нужно нажать:

Конфигурация -> Сохранить конфигурацию в файл
