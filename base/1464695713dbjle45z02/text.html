Для настройки можно использовать команду:

git config --global http.proxy http://proxyuser:proxypwd@proxy.server.com:8080


В этой команде нужно заменить:

  • proxyuser на имя пользователя сервера proxy
  • proxypwd на пароль на прокси-сервере
  • proxy.server.com на URL вашего прокси-сервера
  • 8080 на порт вашего прокси-сервера


Если возникнет необходимость работать напрямую, без прокси-сервера:

git config --global --unset http.proxy

Команда для проверки, какой прокси-сервер настроен в данный момент:

git config --global --get http.proxy
