Простейшее описание класса в Objective C может выглядеть так:

@interface MyClass

@property NSString *stringProperty;

@property NSInteger integerProperty;

@property MyClass *linkedInstance;

@end

Создание экземляра класса обычно выглядит так:

MyClass *myInstance = [[MyClass alloc] init];

Получение свойства - строки данного класса может выглядеть так:

NSString *string = myInstance.stringProperty;
