Все, что нужно знать о корпоративных лицензиях Microsoft OLP


30.04.2011


Программы корпоративного лицензирования – предпочтительный способ приобретения лицензий программного обеспечения Microsoft для организаций. Корпоративные схемы предусматривают значительные скидки по сравнению с розничными ценами и позволяют учесть размер компании и другие особенности бизнеса. Самой популярной в России программой корпоративного лицензирования ПО Microsoft является Open License Program (OLP).

Краткое описание программы

Программа Open License предназначена только для организаций и индивидуальных предпринимателей, она недоступна для частных лиц. Первый заказ должен содержать не менее 5 лицензий. В течение двух лет после размещения первого заказа покупатель может пользоваться полученной корпоративной скидкой и размещать любые по объему заказы, в том числе и на 1 лицензию.

Некоторые покупатели считают, что ограничение в 5 лицензий устанавливает продавец и его можно уговорить на меньшее количество. На самом деле, продавец не имеет в своем распоряжении готовых OLP-лицензий. Они печатаются в Microsoft на специальных бланках с несколькими степенями защиты. Процессом изготовления лицензий управляет компьютерная программа, на вход которой поступает файл с данными, полученными от покупателя. Таким образом, контроль количества лицензий осуществляется автоматически и обойти его невозможно.

Что из себя представляют лицензии OLP?

Microsoft OLP – это именные лицензии, то есть они выписываются на конкретного пользователя (юридическое лицо или индивидуального предпринимателя) и не могут использоваться кем-то другим.

Пользователь получает документ – лицензионный сертификат, содержащий:

  • лицензионное соглашение с пользователем;
  • номера, необходимые для регистрации на специальном сайте Microsoft;
  • сведения о пользователе (наименование компании или ФИО индивидуального предпринимателя, ИНН, юридический адрес);
  • перечень купленных лицензий и их количество.

Лицензионный сертификат поставляется в запечатанном фирменном конверте.

С 10 марта 2013 года сертификаты на бумаге заменяются на электронные.

После размещения заказа Open License в Microsoft пользователю высылается автоматический е-мейл с сайта Microsoft со ссылкой на детали лицензий на сайте Microsoft VLSC. Лицензионный сертификат можно будет просматривать в электронном виде и распечатывать на сайте VLSC.

Какие данные нужны для заказа лицензий OLP?

Для оформления лицензии заказчик должен предоставить Microsoft следующую информацию:

  1. Название организации.
  2. ИНН организации.
  3. Юридический адрес организации (полный, вместе с индексом).
  4. Имя и фамилия контактного лица.
  5. E-mail администратора лицензий (e-mail адрес, через который будет разрешен доступ к ключам и дистрибутивам на сайте Microsoft).
  6. Телефон организации или контактного лица.

Можно ли передать лицензию OLP другому юридическому лицу?

Возможна передача лицензий только аффилированному лицу или неаффилированному третьему лицу в связи с:

  • для лицензий, купленных с 01 октября 2006 до 07 декабря 2009 – присоединением, продажей активов или слиянием аффилированного лица или подразделений;
  • для лицензий, купленных после 7 декабря 2009 – слиянием предприятий или передачей собственности на свое предприятие.

Окончательное решение о возможности передачи лицензий принимает Microsoft. Инструкции по передаче лицензий Open License содержатся в документе Open License Transfer Procedure.

Входит ли диск в поставку OLP?

Носители информации (диски) в состав OLP не входят, они продаются отдельно, в дополнение к лицензиям. Покупка носителей, как правило, не обязательна, так как дистрибутивы программ доступны для скачивания с сайта VLSC.

Срок действия лицензий OLP

Лицензии OLP бессрочные. При выписке лицензии начинает действовать двухлетнее лицензионное соглашение Open License, в течение которого пользователь может размещать дополнительные заказы на лицензии. После истечения срока действия этого соглашения приобретенные лицензии продолжают оставаться в силе.

Срок поставки лицензий OLP

Лицензионные сертификаты печатаются в Ирландии. Время доставки лицензии до офиса в Москве – 5-10 дней с момента размещения заказа.

Типы лицензий OLP

  1. License – полная лицензия на конкретную (языковую и порядковую) версию программного продукта.
  2. Software Assurance (SA) – право перехода на следующие версии продукта, которые будут выпущены в течение срока действия лицензионного соглашения Open License.
  3. License & Software Assurance (LicSAPk) – объединенная позиция (полная лицензия + право перехода на следующие версии продукта).
  4. Upgrade (UPG) – лицензия на обновление (применяется только для Windows 7 Professional).
  5. Upgrade & Software Assurance (UpgrdSAPk) – объединенная позиция (лицензия на обновление + право перехода на следующие версии продукта), которая также дает право использовать Windows 7 Enterprise (применяется только для Windows 7 Professional).

Выбор языка

В OLP предлагается 2 основных варианта выбора языка интерфейса программ:

  • RUS (Russian);
  • SNGL (Single).

Лицензия Single позволяет использовать или русский, или английский язык (точнее, любой из доступных для данного продукта языков) на выбор при установке, лицензия Russian – только русский.

Использование предыдущих версий ПО

Лицензии Open License можно купить только для актуальных версий программ, но они дают право использования и предыдущих версий выбранных редакций ПО.

Например, имея лицензию Office Standard 2010, вы можете использовать Office Standard 2003 и Office Standard 2007. Лицензия на Windows 7 Professional дает право использовать не только предыдущие версии этого продукта, но и Windows 95 или Windows 98.

Особенности лицензий Windows OLP

Изначально в OLP можно было приобрести только лицензию на обновление операционной системы Windows (Upgrade). Лицензии на полную версию продавались вместе с компьютерами (OEM) или в коробочном виде. Однако в связи с ростом интереса пользователей к легальному ПО появилась необходимость в выпуске специальной лицензии "для легализации" – Get Genuine Windows Agreement (GGWA).

Get Genuine Windows Agreement – лицензия для легализации настольных операционных систем на имеющихся компьютерах. В комплект поставки GGWA входит наклейка сертификата подлинности (COA) для каждого ПК. Лицензии GGWA – невозвратные.

Условия использования:

  • На каждый ПК, лицензированный через GGWA, необходимо наклеить сертификат подлинности (COA).
  • Перенос лицензии на другой ПК запрещен.
  • Передача лицензий возможна только в соответствии с условиями соглашения Open License.
  • В течение 90 дней с даты приобретения лицензии разрешается приобретение Software Assurance.
  • Возможно приобретение лицензий GGWA для аффилированных лиц в рамках одного соглашения.
  • Downgrade возможен до предыдущих редакций ОС, включая Windows XP Pro (и вплоть до Windows 95).

Цены и скидки

В OLP есть два уровня цен:

  1. Уровень цен NL – минимум 5 лицензий на любые продукты.
  2. Уровень цен C – минимум 500 баллов в конкретной категории продуктов.

Все продукты разделены на три категории: "Системы", "Приложения", "Серверы". Каждому продукту приписан весовой коэффициент-"балл" (например, 1 лицензия на Office "весит" два балла). Объем заказа в баллах рассчитывается отдельно для каждой группы продуктов.

Фактически это означает, что переход на уровень C становится доступным при заказе 250 лицензий какого-то продукта (например, Office). Скидка по сравнению с NL составляет около 2%.

Уровень скидки, достигнутый при размещении первоначального заказа, сохраняется за пользователем в течение двух лет действия соглашения Microsoft Open License. Для уровня C скидка действительна в пределах категории продуктов. Важно отметить, что фиксируется не цена, а только уровень скидки.

Примеры лицензий OLP

  • WinSvrStd RUS LicSAPk OLP NL (Microsoft Windows Server Standard Russian License/Software Assurance Pack OPEN No Level) – полная лицензия + Software Assurance на Microsoft Windows Server Standard, русская версия, уровень цен NL
  • OfficeStd 2010 SNGL OLP C (Microsoft Office 2010 Sngl OPEN 1 License Level C) – полная лицензия на Microsoft Office 2010 Standard, одноязыковая версия, уровень цен C.

Список программных продуктов, доступных по OLP

Для заказа по программе Open License доступны практически все программные продукты, предназначенные для корпоративного использования. Среди них:

  • Windows
  • Office для Windows и Mac
  • Windows Small Business Server
  • Windows Server
  • Exchange Server
  • SQL Server
  • Lync Server
  • Visual Studio
  • Forefront Threat Management Gateway

Узнать о доступности того или иного продукта можно у менеджеров компании "СофтМарк".

Что делать с лицензиями OLP, как получить ключи и дистрибутивы?

Для покупателей, впервые сталкивающихся с приобретением OLP, как правило, оказывается не очень понятным дальнейшее применение лицензий. На самом деле, лицензионный сертификат содержит всю необходимую информацию для получения доступа к ключам и дистрибутивам.

  1. Дистрибутивы и лицензионные ключи, которые будут доступны вам после регистрации на сайте VLSC.  Для регистрации и входа в персональный раздел сайта используйте авторизационный номер соглашения и номер лицензии OLP.

    2. Для входа на VLSC потребуется Live ID, связанный с адресом e-mail, указанным при размещении заказа. Если у вас его нет, получить его можно здесь: http://www.microsoft.com/rus/liveid/.

    Подробное руководство по использованию сайта VLSC можно посмотреть вот здесь: http://www.microsoft.com/rus/licensing/afterbuying/running/installation/default.aspx и здесь: http://download.microsoft.com/documents/rus/Licensing/doc/Volume+Licensing+Center+User+Guide+-+RU.docx.

  2. Для подтверждения легальности используемого ПО обязательно сохраните полный комплект, включая упаковку, а также копии бухгалтерских документов.
