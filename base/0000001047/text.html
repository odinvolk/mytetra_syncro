No-IP – это программа с помощью которой своему компьютеру вместо обычного IP-адреса можно присвоить постоянное доменное имя.


Можно будет обращаться к своему компьютеру по доменному имени прямо из интернета. При смене IP-адреса, No-IP распространяет изменения вашего IP-адреса по сети, а доменное имя не изменяется.


Ограничение: сервис не будет работать, если компьютер находится за NAT провайдера. А именно так работают практически все провайдеры в России (Ростелеком - точно). Для использования сервиса необходим белый IP.



Кракое описание, как пользоватья клиентом no-ip.com в Linux


Основные команды. Их надо выполнять из-под суперпользователя.


/usr/local/bin/noip2 -C configure a client

/usr/local/bin/noip2 run a client

/usr/local/bin/noip2 -S display info about running clients

/usr/local/bin/noip2 -D pid toggle the debug state for client pid

/usr/local/bin/noip2 -K pid terminate client pid


Что делать, если имеется более чем одно устройства для соединения с сетью Интернет?


/usr/local/bin/noip2 -M -c file


Эта команда создаст отдельное новое соединение.
