На старых версиях K3B невозможно записать на DVD болванку файл, размером больше чем 4 GB.

Решение такое:

  • создаем образ на жестком диске,
  • примонтируем его,
  • копируем туда файл,
  • записываем образ на DVD.

Берем размер образа (в секторах) по следующей таблице:

DVD-R/DL = 4,171,712

DVD+R/DL = 4,173,824

DVD-R/SL = 2,298,496

DVD+R/SL = 2,295,104

Размер сектора, кстати, 2048 байт.

Возмем DVD+R/single layer для примера. Делаем образ:

$ dd if=/dev/zero of=/mnt/data/image.udf bs=2048 count=2295104

Внутри файла /mnt/data/image.udf создаем файловую систему UDF:

mkudffs --lvid=Volume_name /mnt/data/image.udf

Монтирование файла как образ:

sudo mount -o loop /mnt/data/image.udf /mnt/dvdimage
