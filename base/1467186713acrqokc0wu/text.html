В Debian, начиная с версии 7, в настройках SSH-сервера сконфигурирован вход пользователя root только по сертификату. Такое соединение считается безопасным. Поэтому вход в систему под root по SSH невозможен.


Чтобы разрешить вход пользователя root по паролю, нужно сделать следующее.


Открыть файл /etc/ssh/sshd_config.


Найти опцию:


PermitRootLogin without-password


и заменить ее на:


PermitRootLogin yes


После чего надо перезапустить SSH-сервер.
