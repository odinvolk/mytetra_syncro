Программа transcode способна работать с многодорожечными AC3 файлами (например, шесть дорожек в формате 5.1) и способна сконвертировать из такого файла новый стерео WAV - файл (с двумя дорожками). При этом сведение дорожек происходит правильно, в соответсвии с коэффициентами усиления, заданными в AC3 файле для каждой дорожки.

Формат команды следующий:

# transcode -i file_in.ac3 -y null,wav -o file_out.wav
