Перевернуть изображение на 180 градусов можно с помощью комбинаций фильтров flip и mirror.


flip зеркально отображает изображение относительно горизонтали. Но при этом получается "зеркальное" отображение.


mirror зеркально отображает изображение относительно вертикали. И будучи примененным после flip, делает правильное изображение.


Строка с командой будет следующая:


mplayer -vf flip,mirror MOV_0009.AVI


Повернуть видео на 90 градусов можно с помощью фильтра rotate:


mplayer -vf rotate=1 movie_with_wrong_rotation.avi


Выдержка из документации:


ВИДЕО ФИЛЬТРЫ

Видео фильтры позволяют изменять видео поток и его свойства. Синтаксис

такой:


-vf <фильтр1[=параметр1:параметр2:...],фильтр2,...>

Устанавливает цепочку видео фильтров.


rotate[=<0-7>]

Поворачивает изображение на 90 градусов и опционально зеркально

его отражает. Для значений между 4-7 поворот производится

только в том случае, если фильм имеет книжную, а не альбомную

ориентацию.


0 Поворот на 90 градусов по часовой стрелке и отражение

(по умолчанию).


1 Поворот на 90 градусов по часовой стрелке.


2 Поворот на 90 градусов против часовой стрелки.


3 Поворот на 90 градусов против часовой стрелки и

отражение.
