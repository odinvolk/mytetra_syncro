Основные команды следующие:


apt-get update - Обновляется локальный список пакетов, присутсвующих в репозитариях. Обычно нужно запускать эту команду после того, как добавлен новый репозитарий в файл /etc/apt/sources.list.


apt-get install имя_пакета - Установка пакета. Пакет будет браться из репозитария.


dpkg -i пакет.deb - Установка программы из deb-файла.


apt-get remove имя_пакета - Удаление пакета.


apt-get remove --purge имя_пакета - Полное удаление пакета, включая файлы настроек


apt-get upgrade - Обновление системы. Будут установлены самые последние версии пакетов, присутсвующие в репозитариях.


apt-get clean - Удаляет все пакеты из кеша, обычно кеш - это директория /var/cache/apt/archives.
