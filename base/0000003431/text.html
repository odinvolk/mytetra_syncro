Для переконвертирования образов CD дисков из img в iso формат, нужно воспользоваться программой ccd2iso:

$ ccd2iso fle.img file.iso

Если программа вываливается с ошибкой

Unrecognized sector mode (c9) at sector 0!

Тогда нужно проверить, не является ли img-образ iso-образом. Это можно сделать с помощью команды file:

$ file cddisk.img

cddisk.img: ISO 9660 CD-ROM filesystem data 'netbook-ia32-x86_64-20101026155' (bootable)

Еще, для конвертации, можно воспользоваться программой acetoneiso, которая есть в Debian репозитариях.
