Внутри D-Link DSL-300T находится небольшой веб-сервер.


Отправить POST-запрос этому серверу изнутри самой системы, то есть, будучи залогиненым в консоль D-Link DSL-300T, можно следующей командой:


echo "POST /cgi-bin/firmwarecfg HTTP/1.1" | /usr/www/cgi-bin/firmwarecfg


Этот метод может пригодиться для автоматизации каких-нибудь настроечных действий, если их можно производить только через веб-интерфейс администрирования.
